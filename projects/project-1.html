Command Line Mancala

Description:
This was a group project, as part of the final for the course titled: CS 3110: Functional Programming and Data Structures.
Our project is a command-line interactive version of the game Mancala. It supports two-player gameplay and additionally allows users to play against a computer AI in a single-player mode, with four difficulty levels to choose from. The interface keeps track of the winner for each round and provides an option to continue gameplay by prompting to play again.

Skills & Tools

  • Programming Languages: OCaml
  • Tools: Git
  • Other: Agile Workflow Methodology,

Purpose & Use

  • Purpose: This project was created as a final composition for a course on functional programming.
  • Use: This game is primarily for entertainment purposes, as it is a digital version of the traditional Mancala game. Additionally, this can be used to become more familiar with the terminal via game.
